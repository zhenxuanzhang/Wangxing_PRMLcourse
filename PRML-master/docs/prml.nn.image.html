prml.nn.image.deconvolve2d.deconvolve2d(x, y, stride=1, pad=0, shape=None)[source]¶

deconvolution of two tensors aka transposed convolution

Parameters:
  • x ((n_batch, xlen, ylen, in_chaprml.nnel) Tensor) – input tensor to be deconvolved
  • y ((kx, ky, out_chaprml.nnel, in_chaprml.nnel) Tensor) – deconvolution kernel
  • stride (int or tuple of ints (sx, sy)) – stride of kernel application
  • pad (int or tuple of ints (px, py)) – padding image
  • shape (tuple of ints (xlen', ylen')) – desired shape of output image If not specified, the output has the following length len’ = s * (len - 1) - 2p + k
Returns:

output – The first argument deconvolved with the second one len’ will be the following if not specified len’ = s * (len - 1) - 2p + k
Return type:

(n_batch, xlen’, ylen’, out_chaprml.nnel) Tensor
