prml.markov.kalman module¶

class prml.markov.kalman.Kalman(system, cov_system, measure, cov_measure, mu0, P0)[source]¶

Bases: prml.markov.state_space_model.StateSpaceModel

A class to perform kalman filtering or smoothing

\(z\) : internal state (random variable)

\(x\) : observation (random variable)

prior distributions:

\(p(z_0) = \mathcal{N}(\mu_0, P_0)\)

\(p(z_n) = \int p(z_n|z_{n-1})p(z_{n-1}) {\rm d}z_{n-1} = \mathcal{N}(A\mu_{n-1},AP_{n-1}A^{\rm T}+\Gamma) = \mathcal{N}(\mu_n, P_n)\)

\(p(x_n)=\int p(x_n|z_n)p(z_n){\rm d}z_n=\mathcal{N}(C\mu_n,CP_nC^{\rm T}+\Sigma)\)

Parameters:
  • system ((Dz, Dz) np.ndarray) – system matrix aka transition matrix (\(A\))
  • cov_system ((Dz, Dz) np.ndarray) – covariance matrix of process noise (\(\Gamma\))
  • measure ((Dx, Dz) np.ndarray) – measurement matrix aka observation matrix (\(C\))
  • cov_measure ((Dx, Dx) np.ndarray) – covariance matrix of measurement noise (\(\Sigma\))
  • mu0 ((Dz,) np.ndarray) – mean parameter of initial hidden variable (\(\mu_0\))
  • P0 ((Dz, Dz) np.ndarray) – covariance parameter of initial hidden variable (\(P_0\))
